# TG263NameChecker
Python code to check if structure names match TG263 standard

These are examples of CLI commands to run the script. 

For single

python TG263NameCheck.py -type single "Heart_L"

For file not specifying the output file

python TG263NameCheck.py -type file planlist2_part3_output.txt

For file with specifying the output file

python TG263NameCheck.py -type file planlist2_part3_output.txt results.xlsx
